TAGS:

  1. table = Exemplificando:< table > < /table >
    • Usada para defiir uma tabela
    • Apenas a tag table não cria toda tabela
    • É necessário usar as tags tr e td para criar linhas e células respectivamente
    • Uma tabela mais complexa ainda pode conter as tags caption, col, colgroup, thead, tfoot e tbody


  2. del = Exemplificando:< del > < /del >
    • Tag usada para definir um texto que foi excluido ;
    • cite = define a fonte do motivo pelo qual o texto foi excluido ;
    • datetime = Define quando o texto foi excluido;
    Exemplo : Direitos Autorais

  3. ins = Exemplificando:< ins > < /ins >
    • Tag usada para definir um texto que doi inserido ou alterado
    • cite = define a fonte do motivo pelo qual o texto foi excluido ;
    • datetime = Define quando o texto foi excluido;
    Exemplo : Exemplo

  4. sub = Exemplificando:< sub > < /sub >
    • Tag usada para definir um texto subscrito;
    • Normalmente usado para fórmulas quimicas;
    Exemplo : Formula quimica da água é H20

  5. blockquote = Exemplificando:< blackqote > </blockquote >
    • Tag usada para definir uma citação externa
    • Para citações curtas use a tag q
    Exemplo :
    A citação é uma marca do dialogismo linguístico, feita para sustentar uma hipótese, reforçar uma ideia ou ilustrar um raciocínio. Oferece ao leitor respaldo para que possa comprovar a veracidade das informações fornecidas e também possibilitar seu aprofundamento. Ressalva-se que a referência bibliográfica (dados que identificam uma publicação citada, tais como autor, título, editora) deve aparecer no final do trabalho sob o título de "referências", pois, desta maneira, o leitor poderá identificar a obra, facilitando sua localização em catálogos, índices bibliográficos, bibliotecas, Internet, entre outros.
  6. q = Exemplificando:< q > < /q >
    • Usada para definir uma sitação simples
    Exemplo : A citação é uma marca do dialogismo linguístico

  7. abbr = Exemplificando:< abbr > < /abbr >
    • Tag usada para definir uma abreviação.
    • Importante para fornecer informações úteis para navegadores, sistema de tradução e mecanismo de busca.
    Exemplo : Hoje choveu muito na cidade de Rebouças PR

  8. address = Exemplificando:< address > < /addres >
    • Tag usada para definir as informações de contato do autor do documento.
    • Tag pode incluir o endereço fisico, URL, e-mail,telefone, midia social e coordenadas geográficas.
    • Deve incluir o nome da pessoa, grupo ou organização.
    Exemplo:
    Bruno Luciano Przybisz
    Minha loja como jogador : Polako
    Acesse meu canal https://www.youtube.com/channel/UCBGZt7suNlIRiqhi7oShlKg
    Rua Antonio Hermino Moreira de Souza
    10º Andar - Centro
    Rebouças - PR
    84550-000
  9. cite =Exemplificando:< cite > < /cite >
    • Tag usada para definir o titulo de uma obra(Livro, música, filme, etc)
    Exemplo : Faroeste Caboclo música da banda Legião Urbana lançada em 1987.

  10. bdo =Exemplificando:< bdo > < /bdo >
    • Tag usada definir a direção do texto.
    • Essa tag substitui a direção do texto atual
    • O atributo dir é obrigatório
    • dir = dois valores (ltr, rtl)
    Exemplo : Duas coisas são infinitas: o universo e a estupidez humana
    Não tentes ser bem-sucedido, tenta antes ser um homem de valor

  11. code = Exemplificando:< code > < /code >
    • Tag usada para definir parte de um texto como parte de código de computador
    Exemplo : $name = ['Bruno','Luciano'];

  12. pre =Exemplificando:< pre > < /pre >
    • Tag usada para definir um texto pré-formatado
    • Esta tag preserva os espaços e quebras de linha
    Exemplo: 
       
                    Aluno: Bruno Luciano Przybisz
                    Na Trilha Do Aprendizado
  13. kbd = Exemplificando:< kbd > < /kbd >
    • Tag usada para definir uma entrada do teclado em uma frase.
    Exemplo : Para copiar o texto use as teclas CTRL + C
